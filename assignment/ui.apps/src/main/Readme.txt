Calculate Student Total Marks

How to Run the Application

1. Add the StudentMarks Component to a Page.

2. Configure the Component:
   - Click on the StudentMarks component to open its dialog box.

3. Save the Details:

4. Calculate Total Marks:
   - Once the details are saved, you will see an `Add` button on the page.
   - Click the `Add` button to trigger the calculation of the total marks.
   - The total marks will be displayed in the `Total Marks` section below the button.
